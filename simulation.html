About LEAP integration
This simulation uses the LEAP COM API to communicate with the LEAP 25.11.25 desktop application installed on this machine. The Python backend calls win32com.client.Dispatch("LEAP.LEAPApplication") to trigger calculations and retrieve results. LEAP must be installed and the model file must be accessible for the simulation to run.
